Programmable Omni-Directional Robotic Arm Vehicle

Description

This robotic arm vehicle is designed for omnidirectional movement and can be programmed to perform various tasks.

Key Features

  • Omni-directional movement
  • Programmable using various languages
  • Robotic arm with 3-4 DOF
  • Integrated sensors for navigation
  • Durable and compact design
  • Wireless communication
  • Real-time monitoring

Technical Specifications

Dimension 30 cm x 20 cm x 15 cm
Weight 5 kg
Power Supply 12V DC
Programming Language C, C++, Python, etc.
Communication Wireless (Wi-Fi, Bluetooth)
